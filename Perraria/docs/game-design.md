# Perraria v1.0 - 游戏设计文档

## 目标
实现一个可玩的 2D 横版生存建造游戏最小原型。能跑、能挖、能放、能打、能死。

## 核心信息
- **类型**: 2D 横版生存建造
- **模式**: 单人
- **视角**: 2D 侧视
- **美术风格**: 像素风（16x16 或 32x32 像素方块）

## v1.0 核心系统

### 1. 世界
- 单一生态群落（草地+泥土+石头+矿石）
- 程序化地形生成（地表起伏 + 地下洞穴）
- Tilemap 方块系统
- 固定世界尺寸，不做区块动态加载

### 2. 玩家
- 行走 + 跳跃
- 生命值（受击扣血，自然回复）
- 鼠标指向方块进行挖掘/放置
- 鼠标点击攻击敌人

### 3. 物品
- 物品类型：方块、工具（镐）、武器（剑）、材料
- 快捷栏（底部 10 格）
- 背包（基础网格，无分页）
- 方块被挖掘后掉落物品，靠近自动拾取

### 4. 敌人
- 1-2 种基础地面敌人（如史莱姆）
- 简单 AI：朝玩家移动 + 接触伤害
- 被击杀后掉落物品

### 5. UI
- 生命值条
- 快捷栏
- 背包界面（按键切换）
