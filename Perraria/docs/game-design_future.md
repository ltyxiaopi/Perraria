# Perraria - 游戏设计文档

## 游戏概述
- **类型**: 2D 横版生存建造游戏（类 Terraria）
- **模式**: 单人优先
- **视角**: 2D 侧视
- **美术风格**: 待定（像素风 / 卡通风）

## 核心玩法循环
```
探索世界 → 采集资源 → 合成物品 → 建造基地 → 战斗生存 → 深入探索
```

## 核心系统

### 世界
- 程序化地形生成（地表、洞穴、矿脉）
- Tilemap 方块系统，可挖掘/放置
- 昼夜循环
- 多种生态群落（森林、沙漠、雪地、地下）

### 玩家
- 移动：行走、跳跃、游泳
- 生命值、饥饿值（待定）
- 工具使用：镐、斧、剑
- 装备系统

### 物品与合成
- 物品分类：材料、工具、武器、盔甲、消耗品、方块
- 工作台合成系统
- 物品栏 + 快捷栏

### 敌人与战斗
- 近战 / 远程攻击
- 不同生态群落对应不同敌人
- Boss 战（待定）

### 建造
- 方块放置/破坏
- 家具/装饰物
- NPC 入住条件（待定）
